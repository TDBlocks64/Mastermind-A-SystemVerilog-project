Mastermind: A SystemVerilog project

This project is about creating SystemVerilog code that is specifically built to run Mastermind.
This project was developed with the Vivado Design Suite, Basys 3 Artix 7 development board and HP EliteDisplay E232 monitor.
Might not work or need adjustment to work with other design software, development boards or monitors!

Features contained within this project:
- Button debouncer for the input handling.
- Core module containing all the game logic.
- Output module generating timing for 1280x720 @ 60Hz VGA signals.
